# Student Database Management System

Very convenient database management system for managing student entires.

This software is base CRUD application which provides more functionalities then CRUD application.

![Preview](https://i.imgur.com/MZwXK0v.png)

## Used languages

- Python
- SQLite

## Features

- [x] Authentication required for accessing the software.
- [x] Managing entires (Create, Read, Edit, Delete).
- [x] Import data from excel sheet.
- [x] Export data to excel sheet.
- [x] Automatic data backup.
  - [x] Backup as excel sheet.
  - [x] Backup as database file.
- [x] Viewer(Default) and Editor mode.
- [x] Switching to editor mode requires authentication with different credentials.
- [x] Searching Functionality.
  - [x] Search via Columns.
  - [x] Search via Entires.
  - [x] Search Specific Entry.
- [x] Print searched data.
  - [x] Print specific Column.
  - [x] Print multiple selected Column.
- [x] Wipe entire database ⚠️ (Must be used with caution).
